This directory contains demo CSS and template blocks that you can reference from your Twig templates.

Feel free to add more and make any changes.

Files with these names should always exist, even if you're not using them:

       '/custom/pgr/PowerReplacer/custom_styles.css',
       '/custom/pgr/PowerReplacer/custom_header.html',
       '/custom/pgr/PowerReplacer/custom_footer.html'
